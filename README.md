# Парсер расписания занятий

Программа для парсинга PDF файла с расписанием занятий и извлечения данных в формате JSON.

## Установка зависимостей

```bash
pip install pdfplumber
```

или

```bash
pip3 install pdfplumber
```

## Использование

```bash
python3 parse_timetable.py
```

Программа прочитает файл `Lechebnoe delo-13-01-26.pdf` и создаст файл `timetable.json` с результатами парсинга.

## Формат выходных данных

Каждая запись в JSON содержит следующие поля:

- `discipline` - название дисциплины
- `group` - номер группы
- `day_of_week` - день недели (monday, tuesday, wednesday, thursday, friday, saturday, sunday)
- `room` - номер аудитории (например, А436, У606)
- `period` - номер пары (1-6)
- `institute` - название института
- `specialty` - специальность
- `course` - курс (1-6)
- `even_week` - проводится ли занятие в четные недели (true/false)
- `odd_week` - проводится ли занятие в нечетные недели (true/false)
- `subgroup` - номер подгруппы (если есть, иначе null)
- `lesson_type` - тип занятия (lecture, practice, subgroup)
- `period_dates` - период проведения занятий

## Особенности парсинга

- Если в тексте есть разделитель `//`, то первая часть (до `//`) относится к четным неделям, вторая часть (после `//`) - к нечетным неделям
- Если разделителя `//` нет, занятие проводится каждую неделю
- Если строка разделена на несколько колонок, это означает разные подгруппы
- Пустые ячейки означают, что подгруппа не ходит в это время
